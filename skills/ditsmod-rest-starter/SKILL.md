## Installation

```bash
git clone --depth 1 https://github.com/ditsmod/rest-starter.git my-app
cd my-app
npm i
```

Alternatively, you can use the starter monorepo:

```bash
git clone --depth 1 https://github.com/ditsmod/rest-monorepo-starter.git my-app
cd my-app
npm i
```
